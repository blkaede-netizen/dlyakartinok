# Школа Вкусного Бизнеса

Одностраничный информационный сайт компании. Проект использует чистые HTML, CSS и JavaScript без сборщика и внешних библиотек.

## Публикация на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в корень репозитория `index.html`, папку `assets`, файлы `.nojekyll` и `.gitignore`.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main`, папку `/ (root)` и нажмите **Save**.

После публикации сайт будет доступен по адресу:

`https://ИМЯ-ПОЛЬЗОВАТЕЛЯ.github.io/ИМЯ-РЕПОЗИТОРИЯ/`

## Локальный просмотр

Откройте `index.html` в браузере. Все используемые изображения находятся в `assets/`.
